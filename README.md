# Админка управления заказами

Проект — админ панель для администраторов, менеджеров и клиентов. Клиенты могут только просматривать каталог товаров, менеджеры просматривают заказы, администраторы имеют полный доступ — создание/удаление/изменение товаров и заказов.

Проект написан в качестве практической работы по предмету "Методологии Разработки Программного Обеспечения". В данной практической работе реализованы все четыре требуемых модуля.

## Требования

- Python 3.12+
- Docker + Docker Compose

## Установка и запуск

Скопировать и заполнить `.env` корректными данными:
```bash
cp .env.example .env
```

Запуск в Docker-контейнерах:
```bash
docker compose up --build
```

Запуск локально (требуется поднятый PostgreSQL):
```bash
uv sync
source .venv/bin/activate
python manage.py migrate
python manage.py runserver
```

## Стек технологий

- Django 5 + PostgreSQL
- Django-HTMX + Bootstrap 5
- uv (пакетный менеджер)
